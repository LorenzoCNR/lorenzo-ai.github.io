Partial Cointegration & Statistical Arbitrage

This project investigates statistical arbitrage strategies based on partial cointegration, a framework that extends classical cointegration by allowing spreads to contain both a mean-reverting component and a persistent stochastic trend.

The objective is to evaluate whether partial cointegration models provide more robust trading signals than classical Engle–Granger cointegration when applied to equity pairs.

Motivation

Traditional pairs trading assumes that spreads are fully mean-reverting. However, many empirical spreads exhibit hybrid dynamics combining short-term mean reversion with longer-term stochastic drift. Partial cointegration models capture this structure by decomposing the spread into stationary and non-stationary components.

Model Framework

Under partial cointegration the spread can be represented as 

W_t = M_t + R_t

M_t = ρ M_{t−1} + ε_t
R_t = R_{t−1} + η_t

where:

  • M_t is a mean-reverting AR(1) component
  • R_t is a random walk component
  • W_t is the observed spread

The latent components are estimated through a Kalman filtering approach.

Data & Pair Selection

  • S&P 500 equity universe
  • Daily adjusted prices
  • Correlation-based pair screening
  • Cointegration diagnostics

Trading Strategy

Trading signals are generated from the standardized spread: 

z_t = (spread_t − μ) / σ
  • Enter positions when |z| exceeds threshold
  • Exit when spread reverts toward equilibrium
  • Transaction costs and slippage included

Model Comparison

The project compares two modeling frameworks:

  • Engle–Granger Cointegration
  • Partial Cointegration (state-space representation)

Performance is evaluated through cumulative returns, Sharpe ratio and drawdown statistics.

Implementation

The full pipeline is implemented in R using econometric time-series libraries for cointegration testing, state-space estimation and financial performance analysis.
